# Recode Townsend deprivation index (TDI) https://biobank.ndph.ox.ac.uk/showcase/field.cgi?id=22189

Recode Townsend deprivation index (TDI)
https://biobank.ndph.ox.ac.uk/showcase/field.cgi?id=22189

## Usage

``` r
leo_tdi(tdi, ...)
```

## Arguments

- tdi:

  Numeric/character vector, usually from p22189.

## Value

Numeric vector, with UKB special codes -3/-1 recoded to NA.
