# Save Table 1 to Word or CSV

This function saves the Table 1 output as a Word or CSV file, depending
on the file extension.

## Usage

``` r
leo.table1.save(table, filename)
```

## Arguments

- table:

  A table object (e.g., from CreateTableOne).

- filename:

  A character string indicating the output file name (e.g., "table1.csv"
  or "table1.docx").
