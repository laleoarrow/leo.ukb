# Recode ethnicity to White vs non-White https://biobank.ndph.ox.ac.uk/showcase/field.cgi?id=21000

Recode ethnicity to White vs non-White
https://biobank.ndph.ox.ac.uk/showcase/field.cgi?id=21000

## Usage

``` r
leo_ethnicity(ethnicity, ...)
```

## Arguments

- ethnicity:

  Numeric/character vector, usually from p21000.

## Value

Factor with levels "White", "Non-White"; NA for -1/-3/missing.
